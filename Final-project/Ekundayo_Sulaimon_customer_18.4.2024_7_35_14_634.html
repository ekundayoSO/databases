1 Create a folder for the project

Create a folder for the project. Name the folder with Ekundayo_Sulaimon_customer_project.

2 Write create statements for the database

Write create statements to create a database called customerdb.
Create a table customer, for customer data. Table columns are:
  • customerId integer
  • firstname varchar(11)
  • lastname varchar(6)
  • address varchar(12)
  • customerclass varchar(28)
All columns are mandatory. The primary key is customerId.
Add two insert statements into the create statements for testing.
Create user lucas@localhost with password unfGIN2Q. Grant all necessary privileges to the user.
Save all necessary create statements into a sql file Ekundayo_Sulaimon_customer_createStatements.sql

3 Create the database

Create the database with the create statements .

4 Write Sql statements

After each operation check the content of the database by selecting all data from the table.
Write sql statements, with which you can insert, update, delete and select the customer's data.
Save sql statements into project folder as a sql file named Ekundayo_Sulaimon_customer_sqlStatements.sql.
